# Bike Sales Dashboard in Excel

An interactive dashboard built in Microsoft Excel to analyze bike sales data. This project visualizes key metrics like sales by region, customer demographics, and product performance using pivot tables, charts, and slicers.

## Features
- **Data Cleaning & Analysis**: Processed raw sales data into insights (e.g., top-selling bikes, revenue trends).
- **Visualizations**: Dynamic charts (bar, line, pie) and slicers for filtering by date, region, age group, etc.
- **Key Insights**: 
  -Middle-aged customers purchase the most bikes.
  -Married individuals show higher bike ownership than singles.
  -Males have a slightly higher average income than females.
  -Bike purchases are highest among customers with shorter commute distances.
  -Graduate and bachelor degree holders are more likely to buy bikes.
  -Peak buying age lies between 30–45 years.

## How to Use
1. Download/clone this repo.
2. Open `Bike-sales-dashboard.xlsx` in Microsoft Excel (version 2016+ recommended for full interactivity).
3. Use the slicers (top-right) to filter data dynamically.

## Tech Stack
- **Tool**: Microsoft Excel (PivotTables, Charts, Slicers).
- **Data**: 1,000+ rows of anonymized bike sales records.
